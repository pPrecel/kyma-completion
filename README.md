# kyma-zsh
![](kyma-zsh.gif)


## Installation

1. Run command: 
``` bash
  git clone https://github.com/pPrecel/kyma-zsh && cp -r "kyma-zsh" "$ZSH/custom/plugins" && rm -rf "kyma-zsh"
```

2. Edit your `.zshrc` file and add the `kyma-zsh` to the `plugins` section

Like this: 
```diff
plugins=(
  # ... your other plugins
+ kyma-zsh
)
```
